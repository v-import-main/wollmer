With this plugin activated, Flamingo will not add any data to its address book.

== Description ==

The Flamingo plugin by Takayuki Miyoshi does save all email addresses entered into CF7 forms. If you don't want or need that, you can disable this function by activating my plugin.

== Installation ==

= From WordPress Plugin Directory =

1. In your backend, go to *Plugins &rarr; Add New* to search for Disable Flamingo Addressbook in the WordPress Plugin Directory.
2. Click *Install Now* and then *Activate*.
3. You're done.

= Manual Installation =

1. Download the plugin's ZIP file to your desktop and extract the archive.
2. Upload the folder *disable-flamingo-addressbook* into the directory `/wp-content/plugins/` on your webserver.
3. Activate *Disable Flamingo Addressbook* via the *Plugins* menu in your WordPress Dashboard.

== Frequently Asked Questions ==

= Does activating this plugin affect existing entries in the Flamingo addressbook? =

No, it does not.
